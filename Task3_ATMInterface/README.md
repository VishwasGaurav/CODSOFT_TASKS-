# ATM Interface

## Description
A Java Console-based ATM Interface developed using Object-Oriented Programming concepts.

## Features
- Check Balance
- Deposit Money
- Withdraw Money
- Input Validation
- Menu Driven Interface
- OOP Concepts (Class, Object, Encapsulation)

## Files
- BankAccount.java
- ATM.java
- Main.java

## Language
Java

## IDE
VS Code / IntelliJ IDEA / Eclipse
